# Mobile2DGame
우송대학교에서 열린 교내 공모전을 참여하기위해 제작한 2D 모바일 게임이다. 케릭터 수집형 RPG 장르의 모바일 게임.

## 팀원

 그래픽     - 김유진  
 그래픽     - 김정훈

 - 유니티 , C#
